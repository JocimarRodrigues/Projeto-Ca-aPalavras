## Projeto em React

- Neste projeto eu criei um site interativo em que o usuário pode jogar um jogo de caça palavras. :zap:
- O usuário tem 3 tentativas para adivinhar a palavra. :zap:
- Ao acertar o usuário ganha 100 pontos. :zap:
